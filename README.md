# أبطال القراءة – Reading Heroes

مبادرة تعليمية تفاعلية لتطوير مهارات القراءة والفهم القرائي لطلاب الصف السادس الابتدائي، استعدادًا لاختبارات **نافس** الوطنية.

## المميزات

- **15 مهارة قرائية** مرتبطة بمعايير نافس
- **3 أوضاع لعب**: التحدي اليومي، التدريب الحر، التدريب على مهارة محددة
- **أصوات تفاعلية** عند الإجابة (مع خيار الكتم)
- **نصيحة يومية** تتغير حسب توقيت الرياض
- **لوحة تقدم** بالمهارات والشارات ونقاط XP
- **لوحة معلم** مع تصدير CSV وحماية بكلمة مرور مشفرة
- **شهادة إنجاز** تُولّد عبر Canvas API (PNG موثوق)
- **تصميم متجاوب** يعمل على الجوال والكمبيوتر

## التشغيل محليًا

يحتاج المشروع لخادم محلي بسبب استخدام ES Modules:

```bash
# باستخدام VS Code Live Server
# 1. افتح المجلد في VS Code
# 2. ثبّت إضافة Live Server
# 3. اضغط "Go Live"

# أو باستخدام Python
python3 -m http.server 8080

# أو باستخدام Node.js
npx serve .
```

## النشر على GitHub Pages

1. أنشئ مستودع (Repository) جديد على GitHub
2. ارفع جميع الملفات إلى الفرع `main`
3. اذهب إلى **Settings → Pages**
4. اختر المصدر: **Deploy from a branch → main → / (root)**
5. انتظر دقيقة ثم افتح الرابط

## ملاحظات للمعلم

### إضافة نصوص جديدة

افتح ملف `data/texts.json` وأضف نصًا جديدًا بالصيغة التالية:

```json
{
  "id": "T016",
  "title": "عنوان النص",
  "genre": "معلوماتي",
  "difficulty": "سهل",
  "body": "محتوى النص هنا...",
  "meta": {"type": "نثر", "genre": "معلوماتي", "topic": "الموضوع"},
  "questions": [
    {
      "id": "T016Q01",
      "skill_id": 1,
      "skill_name": "مرادفات المفردات",
      "stem": "السؤال هنا",
      "choices": ["خيار أ", "خيار ب", "خيار ج", "خيار د"],
      "correct_index": 1,
      "explanation": "شرح الإجابة"
    }
  ]
}
```

### توسيم الأسئلة بالمهارة

كل سؤال يجب أن يحتوي على `skill_id` (رقم من 1 إلى 15):

| الرقم | المهارة |
|-------|---------|
| 1 | مرادفات المفردات |
| 2 | المفردات المتشابهة |
| 3 | المترادفات والأضداد |
| 4 | توظيف المفردات |
| 5 | الفهم المباشر |
| 6 | المقارنة والتحليل |
| 7 | الأفكار الرئيسة والفرعية |
| 8 | الشخصيات والأحداث |
| 9 | الربط بالواقع |
| 10 | التعبيرات الجمالية |
| 11 | وضوح المعلومات |
| 12 | القيم والاتجاهات |
| 13 | العنوان والصياغة البديلة |
| 14 | الإقناع والتعليل |
| 15 | توظيف المغزى |

### لوحة المعلم

- الوصول: `#teacher` في شريط العنوان
- أول مرة: ستطلب منك إعداد كلمة مرور
- كلمة المرور تُخزّن مشفرة (SHA-256 + salt) في LocalStorage
- **تنبيه**: هذه حماية محلية وليست أمانًا كاملًا

### تصدير CSV

- زر "تصدير CSV" في لوحة المعلم
- البيانات معقّمة ضد حقن الصيغ (Formula Injection)
- يشمل ملف UTF-8 BOM لدعم Excel العربي
